AI模型API密钥获取指南

🇨🇳 国内免费模型

百度文心一格

百度推出的AI绘画平台，免费额度丰富

  1. 访问 百度智能云控制台
  2. 注册并登录百度账号
  3. 搜索"文心一格"或"ERNIE-ViLG"
  4. 开通服务后，在应用列表中创建应用
  5. 获取 API Key 和 Secret Key
免费额度: 每月1000次调用
阿里通义万相

阿里云推出的多模态AI平台

  1. 访问 DashScope控制台
  2. 注册阿里云账号并实名认证
  3. 开通DashScope服务
  4. 在API-KEY管理中创建密钥
  5. 复制生成的API Key
免费额度: 新用户有一定免费额度
智谱AI CogView

智谱AI推出的文生图模型

  1. 访问 智谱AI开放平台
  2. 注册并登录账号
  3. 完成个人认证
  4. 在API密钥页面创建新密钥
  5. 复制生成的API Key
免费额度: 注册送免费token
MiniMax文生图

MiniMax推出的AI创作平台

  1. 访问 MiniMax开放平台
  2. 注册并登录账号
  3. 在控制台创建应用
  4. 获取API Key
免费额度: 新用户免费试用

🌍 国外付费模型

OpenAI DALL-E 3

目前最先进的AI绘画模型之一

  1. 访问 OpenAI Platform
  2. 注册OpenAI账号
  3. 绑定信用卡（需要海外支付方式）
  4. 在API Keys页面创建新密钥
  5. 复制生成的API Key
付费: 按使用量计费，1024x1024图片约$0.04/张
Stability AI

Stable Diffusion的官方API服务

  1. 访问 Stability AI Platform
  2. 注册账号
  3. 绑定支付方式
  4. 在API Keys页面创建密钥
付费: 按使用量计费

📖 开源免费模型

Hugging Face

开源AI模型平台，完全免费

  1. 访问 Hugging Face
  2. 注册账号
  3. 在Settings > Access Tokens中创建token
  4. 选择"read"权限即可
完全免费: 无使用限制，但可能有速率限制
Replicate

开源模型运行平台，有免费额度

  1. 访问 Replicate
  2. 注册账号
  3. 在Account settings中找到API tokens
  4. 创建新的API token
免费额度: 每月有一定免费运行时间

💡 使用提示

推荐顺序
  1. 智谱AI - 注册简单，额度充足
  2. 百度文心一格 - 国内稳定
  3. 阿里通义万相 - 质量较高
  4. Hugging Face - 完全免费
注意事项
  • 国内模型无需翻墙，速度更快
  • API密钥请妥善保管，不要泄露
  • 建议先用免费额度测试效果
  • 可以配置多个模型备用
